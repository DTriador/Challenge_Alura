# Análisis de Evasión de Clientes (Churn) - TelecomX

## Descripción
Este proyecto realiza un **análisis exploratorio y predictivo de la evasión de clientes (Churn)** para TelecomX.  
El objetivo es identificar patrones de comportamiento que indiquen la probabilidad de que un cliente cancele el servicio, y generar insights para estrategias de retención.

Se incluyen pasos de limpieza de datos, creación de variables derivadas, análisis exploratorio de variables categóricas y numéricas, y visualizaciones que apoyan la interpretación de los resultados.

---

## Contenido del Proyecto
- `TelecomX_Data.json`: Archivo de datos original con información de clientes.
- `Churn_Analysis.ipynb`: Notebook principal con todo el análisis, visualizaciones y conclusiones.
- `README.md`: Este archivo, que explica el proyecto, instalación y uso.
